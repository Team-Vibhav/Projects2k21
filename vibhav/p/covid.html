Description :

Covid -19 has affected the whole world and it has become really tough to control the outspread of the virus. Keeping the present scenario in mind, we have made a project that can be proved helpful in tackling this situation.Our project is an Automatic Device used to detect mask ,measure the human body temperature and sanitize hands.This project will reduce the contact as well as extra labour needed for the whole process. The main benefit of the project is that it reduces the contact needed for checking the temperature of the person as it will replace the person with a machine. Also,it can perform multiple operations that makes it cost efficient in the present pandemic conditions.
